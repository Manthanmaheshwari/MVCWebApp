# Inventory Management System

A robust and secure web-based Inventory Management System built using the **ASP.NET Core MVC** framework. This application allows businesses or individuals to easily track products, monitor stock levels, and manage inventory data through an intuitive user interface.

##  Features

*   **User Authentication & Authorization:** Secure User Registration and Login functionality to protect inventory data.
*   **Product Management (CRUD Operations):**
    *   **Add Item:** Insert new products along with their names, prices, and available stock.
    *   **View Inventory:** Display a clean, organized list of all tracked items.
    *   **Update Stock:** Modify details or adjust stock numbers as inventory changes.
    *   **Delete Item:** Remove items from the system permanently when they are no longer needed.
*   **Responsive UI:** Styled using Bootstrap to ensure smooth operation across desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

*   **Backend & Framework:** ASP.NET Core MVC (.NET 8.0 / .NET 9.0)
*   **Frontend:** Razor Views, HTML5, CSS3, Bootstrap, JavaScript
*   **Database:** Entity Framework Core (SQL Server / SQLite)
*   **Authentication:** ASP.NET Core Identity
